# AWS S3 Angular Deployment

## Description
Deployed an Angular application using AWS S3 static website hosting.

## Steps
- Built Angular app using ng build
- Uploaded files to S3 bucket
- Configured bucket policy for public access
- Enabled static website hosting

## Result
Live website hosted on AWS S3
